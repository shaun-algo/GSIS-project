CAPSTONE RESEARCH 1
60 PERCENT DEMO GUIDE - SIMPLE PRESENTER SCRIPT
Project: DepEd Academic Monitoring System

Purpose:
This script helps you execute a clean demo flow with simple speaking lines, expected outputs, and code evidence.

==================================================
PART 1. FAST PREP BEFORE START (5 MINUTES)
==================================================
1) Use localhost URL only:
   http://localhost/deped_capstone2/

2) Prepare accounts:
   - Admin
   - Registrar or Teacher
   - Learner (optional)

3) Open browser tabs in this order:
   - index.html
   - pages/enrollments.html
   - pages/grades.html
   - pages/risk-assessment.html
   - pages/report-cards.html
   - pages/profile-security.html

4) Keep DevTools Network tab open to show API responses.
