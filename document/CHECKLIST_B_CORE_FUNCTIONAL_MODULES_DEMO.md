## GSIS-2 Mandatory Feature Check: Early Risk Detection Module

Based on GSIS-2 Appendix D and the "Integration into Early Risk Detection" section, the mandatory rule-based flags are:
- Quarterly Grade below 75
- Final Grade below 75
- General Average below 75
- Continuous grade decline across two or more quarters

### Early Detection Positioning (Important for Defense)

This module is implemented as an **early detection / early warning** feature, not a purely end-of-year classifier.

- It computes and stores risk assessments per grading period, then surfaces the latest period per learner for timely action.
- It can flag learners even before final outcomes are complete, using quarterly performance, trend decline, attendance behavior, and grade completeness indicators.
- It sends risk notifications for non-low risk learners in the latest period to support immediate adviser/admin intervention.

### Centralized Early Detection Flow (Risk Assessment)

Yes. The implementation follows a centralized early-detection pipeline in one orchestrated backend flow.

1. Data is centralized from core modules (enrollment, grades, attendance, periods, sections) and processed through a single automation entry point.

3. Final grades and general averages are then computed in the same run.

### Formula Basis (DepEd-Aligned Rule Computation)

1. Initial Grade (IG): weighted sum of components

$$
IG = (WW \times w_{WW}) + (PT \times w_{PT}) + (QE \times w_{QE})
$$

Where $w_{WW}, w_{PT}, w_{QE}$ are curriculum-defined weights (fallback: 30/50/20 when not configured).

2. Quarterly Grade (QG): transmuted from IG using piecewise mapping

$$
QG =
\begin{cases}
60 + \left(\frac{IG}{60}\right) \times 15, & IG \le 60 \\
75 + \left(\frac{IG - 60}{40}\right) \times 25, & IG > 60
\end{cases}
$$

3. Final Grade (FG): average of quarterly grades per class/subject once all expected periods are encoded.

$$
FG = \frac{\sum QG_{period}}{N_{periods}}
$$

4. General Average (GA): average of final grades across enrolled classes.

$$
GA = \frac{\sum FG_{subjects}}{N_{subjects}}
$$

- **Grade Computation**: [api/grades/grades.php](api/grades/grades.php#L150-L200)
  ```php
  function computeInitialGrade(?float $ww, ?float $pt, ?float $qe, array $weights): ?float {
      if ($ww === null || $pt === null || $qe === null) return null;
      $initial = ($ww * (($weights['WW'] ?? 30.0) / 100.0))
          + ($pt * (($weights['PT'] ?? 50.0) / 100.0))
          + ($qe * (($weights['QE'] ?? 20.0) / 100.0));
      return round($initial, 2);
  }
  ```

- **Transmutation Logic**: [api/grades/grades.php](api/grades/grades.php#L110-L130)
  ```php
  function transmuteGrade(?float $initial): ?float {
      if ($initial === null) return null;
      $g = max(0.0, min(100.0, (float)$initial));
      if ($g <= 60.0) {
          $trans = 60.0 + ($g / 60.0) * 15.0;
      } else {
          $trans = 75.0 + (($g - 60.0) / 40.0) * 25.0;
      }
      return round(max(60.0, min(100.0, $trans)), 2);
  }
  ```
